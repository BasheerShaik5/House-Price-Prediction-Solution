# House Price Prediction using Machine Learning

## Project Overview

This project focuses on predicting residential property sale prices using machine learning techniques. The workflow includes data cleaning, exploratory data analysis (EDA), feature engineering, and regression modeling to identify factors influencing house prices and generate accurate predictions.

## Dataset

- Source: Kaggle House Prices Dataset
- Records: 1,460 residential properties
- Features: 80+ property-related attributes

## Objectives

- Clean and preprocess housing data
- Handle missing values and outliers
- Perform exploratory data analysis (EDA)
- Build predictive models for house price estimation
- Evaluate model performance using regression metrics

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow

### 1. Data Cleaning

- Treated missing values
- Handled outliers
- Prepared data for modeling

### 2. Exploratory Data Analysis (EDA)

- Analyzed feature distributions
- Studied correlations between variables
- Identified key factors affecting house prices

### 3. Feature Engineering

- Encoded categorical variables
- Prepared model-ready features
- Improved dataset quality for training

### 4. Predictive Modeling

Models explored:

- Linear Regression
- Ridge Regression
- Lasso Regression

### 5. Model Evaluation

Model performance was evaluated using regression metrics such as RMSE and RMSLE.

## Key Insights

- Overall property quality strongly influenced sale prices.
- Living area and garage capacity showed significant relationships with house value.
- Proper handling of missing data improved prediction reliability.
